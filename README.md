# WeatherApp iOS 15.0+

WeatherApp позволяет получать прогноз погоды на основе местоположения пользователя или поиска по названию города.

## Описание

WeatherApp отображает текущую температуру, влажность и скорость ветра, почасовой и недельный прогноз. Данные о погоде - weatherapi.com.

## Установка

1. **Сборка проекта**  
   Сборка проекта осуществляется с помощью скрипта `sh.generate`. Выполните следующую команду в терминале из корневой директории:
   ```bash
   sh generate
   ```
   
2. **Настройка API-ключа**  
   Для работы с [Weather API](https://www.weatherapi.com) необходимо получить API-ключ. Сгенерируйте его на сайте и вставьте его в файл `dev.xcconfig`:
   - Откройте файл `dev.xcconfig` в директории проекта WeatherApp/Configs.
   - Найдите строку `WEATHER_API_KEY = ` и замените значение на ваш API-ключ, например:
     ```
     WEATHER_API_KEY = XXX
     ```
